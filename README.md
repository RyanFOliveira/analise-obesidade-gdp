# 🔎 Etapas da Análise

O projeto foi desenvolvido seguindo as seguintes etapas:

### 1️⃣ Carregamento dos dados
Importação dos datasets de obesidade e PIB per capita.

### 2️⃣ Limpeza e preparação
Tratamento de dados, conversão de tipos e organização das tabelas para análise.

### 3️⃣ Análise exploratória
Exploração dos dados para identificar padrões e tendências.

### 4️⃣ Resolução dos desafios analíticos
Resposta às perguntas propostas utilizando técnicas de manipulação de dados com Pandas.

### 5️⃣ Visualização de dados
Criação de gráficos para facilitar a interpretação dos resultados.

---

# 📊 Exemplos de Análises Realizadas

Entre as análises realizadas no projeto estão:

- Identificação do **primeiro registro histórico de PIB por país**
- Ranking de países com **maior PIB per capita**
- Cálculo de **crescimento econômico ao longo do tempo**
- Análise da **evolução da obesidade mundial**

---

# 💡 Principais Insights

A análise dos dados revela alguns padrões interessantes:

- Países desenvolvidos tendem a apresentar maiores valores de PIB per capita.
- O crescimento econômico varia significativamente entre diferentes regiões do mundo.
- As taxas de obesidade têm aumentado ao longo das últimas décadas em diversos países.
